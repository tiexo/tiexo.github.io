原理

百度网盘的自定义密码是在本地生成的
因此采用 JS 的方式修改 makePrivatePassword 这一函数即可

使用

将代码保存为书签网址，名字随意

按照正常的分享步骤，依次完成①和②

接着点击书签，然后再完成③，切记不能乱了顺序
即①-②-书签-③，页面会自动弹出窗口

注意只能输入4位字符，即四个字母或四个数字或一个中文+一个数字或字母

  • test
  • 1234
  • te34
  • 唐1
  • 门t
